  • Lots of git notes to put here
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git pull: command to recieve a branches modifications onto my local computer
  • pwd: print working directory (displays current working directory ((use to confirm the directopry you are in))
  • Ls: shows a list of files in current directory
  • cd: Change directory (apply filename?)

  • touch: creates a file inside of current directory (eg. touch newtxtfile.txt)
  • start: opens application or file (eg. start newtxtfile.txt)
  • code .: opens VScode (when your in applicable directory)
  • git commit -m "reference message goes here with quotations": pushes your version of code to github for merging (has not yet been merged)
  • git add -A: adds all files and changes to repo branch
